Обо мне

Около года назад я принял решение сменить карьеру инженера и попробовать свои силы в веб-разработке. На пути самообучения занимался на онлайн-курсах: "Harvard's CS50 - Introduction to Computer Science", "The Odin Project", "freeCodeCamp". За время обучения поработал с различными технологиями и разработал несколько собственных веб-сайтов и приложений.

Я работаю со следующими технологиями:

  • JavaScript
  • React
  • HTML
  • Webpack
  • CSS
  • Bootstrap
  • Git / GitHub

Также на базовом уровне знаком с Python, SQLite, Flask, jQuery, Sass.

Я стремлюсь к ясному структурированному коду, минималистичному адаптивному дизайну. Получаю удовольствие, шаг за шагом решая комплексные проблемы в процессе разработки.
